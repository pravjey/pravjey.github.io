Redemption Care Services

I am in the process of helping a start-up care provider, Redemption Care Services, to set up a website, including requirements elicitation and web development. I undertook this project as a web developer for Oxyderkis.

The website is being developed primarily using HTML and CSS, with some Javascript for creating a responsive navigation menu and PHP for processing form data and creating automatic acknowledgement emails.

Version 1 was developed following initial requirements' elicitation with the client.

Version 2 was developed following feedback fron the client.

The site is not yet live.
